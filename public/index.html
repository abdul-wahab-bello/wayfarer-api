Sign Up

Sign Up User

Sign up a new user.

  • first_name (string) - User's first name (required).

  • last_name (string) - User's last name (required).

  • email (string) - User's email (required, unique).

  • password (string) - User's password (required).

REQUEST

POST

https://wayfarer-api-001.herokuapp.com/auth/signup
Headers
Content-Type application/json
application/json
BODY SCHEMA 
                        {
    "first_name": "John",
    "last_name": "Doe",
